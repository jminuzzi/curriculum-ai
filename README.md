# Curriculum AI

Sistema inteligente para otimização de currículos com automação e IA.

## Funcionalidades

* Análise automática de currículo
* Otimização baseada em IA
* Automação de candidaturas
* Integração com plataformas de emprego

## Tecnologias

* Python
* FastAPI
* Automation
* AI Integration

## Objetivo

Criar uma solução que ajude candidatos a melhorar currículos e automatizar processos seletivos.
